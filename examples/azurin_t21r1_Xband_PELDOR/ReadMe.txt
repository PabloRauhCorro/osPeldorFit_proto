SPIN SYSTEM
--------------------------------------
Nitroxide-copper(II) spin pair in MTSL-labeled azurin mutant T21R1
(see the reference below).


PDS DATASET
--------------------------------------
Four X-band PELDOR time traces.

References:
D. Abdullin, N. Florin, G. Hagelueken, O. Schiemann,
Angew. Chem. Int. Ed. 2015, 54, 1827-1831.
